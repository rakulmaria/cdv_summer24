How to read the data?

Total annual rain or snow percipation. The bigger the circle, the higher the percipation. Ranges from 1055 - 2084 PP.

Average annual temperature. Lower temperature is blue, higher temperature is red. Ranges from 5.4 - 7.4°C.

Number of days with storm. Each yellow circle represents a day with storm. Ranges from 0 - 5 days.

Number of days with hail. Each light-blue line represents a day with hail. Ranges from 1 - 33 days.

Number of days with snow. Each white dot represents a day with snow. Ranges from 7 - 93 days.

Number of days with fog. The bigger the cloud, the more foggy days. Rotation of cloud is random. Ranges from 20 - 59 days.

Data retrieved from XYZ dataset.
